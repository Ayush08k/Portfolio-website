# Cosmic Amethyst Developer IDE & OS Design System

This design system defines a complete, revolutionary overhaul of the personal portfolio of Ayush Kumar. Rather than a traditional scrolling webpage, the entire site is transformed into an interactive, high-fidelity developer workspace interface (a customized Web IDE/OS Dashboard) utilizing an ultra-luxurious cosmic theme.

## 1. Visual Layout & Structure
- **Screen Canvas**: Stretches full-width and full-height (`100vh`) with a clean, non-scrollable base (contents scroll independently inside the editor canvas).
- **Workspace Border**: Translucent amethyst borders `1px solid rgba(217, 70, 239, 0.12)` outline the window modules to perfectly capture the glowing neon aesthetic.
- **Top Task Bar**: macOS style traffic light window controls on the left, open tabs in the center, system clock & network speed on the right.
- **Left Sidebar**: File Tree Explorer with directory expanding/collapsing controls.
- **Editor Canvas**: Syntactical coding background with modern code line numbers and reactive content.
- **Integrated Mock Terminal**: Bottom command-line drawer that accepts user input commands (`help`, `about`, `clear`, etc.).

## 2. Cosmic Colors & High-Luminance Accents
- **Canvas Base / primary**: `#070312` (Cosmic Black-Purple)
- **Sidebar Background**: `#0b071a` (Warm Deep Amethyst)
- **Editor / Secondary Background**: `#0f0a24` (Amethyst Slate)
- **Accent High-Luminance Cyber Teal/Cyan**: `#00f2fe`
- **Accent Electric Purple/Fuchsia**: `#d946ef`
- **Terminal Green**: `#10b981` (Uptime success indicator)

## 3. Ambient mesh glowing details
- Dual large mesh glowing modules:
  - **Radial Glow 1**: `rgba(217, 70, 239, 0.15)` (Fuchsia aura) on the top right.
  - **Radial Glow 2**: `rgba(0, 242, 254, 0.12)` (Teal aura) on the bottom left.
